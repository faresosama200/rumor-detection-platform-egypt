# 🛡️ منصة مواجهة الشائعات وحروب الجيل الباردة

منصة متقدمة لمواجهة الشائعات والمعلومات المضللة مع نظام ذكاء اصطناعي متطور للتحقق من الأخبار.

## ✨ المميزات الرئيسية

### 🔐 نظام المستخدمين
- **تسجيل الدخول والإنشاء**: نظام آمن للمستخدمين مع أدوار متعددة
- **مدير النظام**: لوحة تحكم خاصة بالإدارة
- **المراجعون**: صلاحيات للتحقق من البلاغات
- **المستخدمون العاديون**: واجهة مبسطة لرفع البلاغات

### 🤖 الذكاء الاصطناعي
- **محادثة ذكية**: مساعد AI متطور للرد على الاستفسارات
- **التحقق الآلي**: تحليل أولي للشائعات
- **تحليل المحتوى**: كشف المعلومات المضللة

### 📊 لوحة التحكم
- **إحصائيات مباشرة**: متابعة البلاغات والتحقق منها
- **رسوم بيانية**: تصورات تفاعلية للبيانات
- **تقارير مفصلة**: تحليلات شاملة للشائعات

### 🎨 التصميم
- **مظهر أسود وذهبي**: تصميم عصري وأنيق
- **واجهة متجاوبة**: تعمل على جميع الأجهزة
- **رسوم متحركة سلسة**: تجربة مستخدم ممتازة

## 🚀 البدء

### المتطلبات
- Node.js (v16 أو أحدث)
- MySQL
- npm أو yarn

### التثبيت

#### 1. تثبيت الخادم الخلفي
```bash
cd fake-news-backend
npm install
```

#### 2. إعداد قاعدة البيانات
```bash
npm run db:init
```

#### 3. إعداد متغيرات البيئة
أنشئ ملف `.env` في مجلد `fake-news-backend`:
```
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=fake_news_db
```

#### 4. تشغيل الخادم الخلفي
```bash
npm run dev
```

#### 5. تثبيت وتشغيل الواجهة الأمامية
```bash
cd frontend
npm install
npm run dev
```

### للمديرين
1. **لوحة التحكم**: إدارة المستخدمين والبلاغات

## 🛠️ التقنيات المستخدمة

### الواجهة الأمامية
- **React 18**: إطار العمل الرئيسي
- **React Router**: التوجيه بين الصفحات
- **Bootstrap 5**: إطار العمل التصميمي
- **Vite**: أداة البناء السريع

### الخادم الخلفي
- **Node.js**: بيئة التشغيل
- **Express.js**: إطار العمل الخلفي
- **MySQL**: قاعدة البيانات
- **JWT**: المصادقة والتفويض
- **Multer**: رفع الملفات
- **bcryptjs**: تشفير كلمات المرور
