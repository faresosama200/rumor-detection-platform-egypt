### 💾 4. جداول قاعدة البيانات الجديدة

**جدول `report_reviews`:**
```

- report_id (ربط مع البلاغ)
- reviewer_id, reviewer_status, reviewer_comment
- admin_id, admin_status, admin_comment
- spokesperson_id, spokesperson_status, spokesperson_comment
- final_verdict (النتيجة النهائية)
- created_at, updated_at
```

**جدول `external_links_config`:**
```
- link_id (المفتاح الأساسي)
- name, url, description
- category, link_type, icon
- is_active (للتفعيل/التعطيل)
```

**تحديث جدول `users`:**
```
- إضافة دور جديد: "spokesperson"
```

### 🔌 5. API Endpoints الجديدة

**البلاغات والمراجعات:**
- `GET /api/reports-with-reviews` - عرض البلاغات مع المراجعات
- `POST /api/reports/:id/review` - إنشاء مراجعة جديدة
- `PUT /api/reports/:id/reviewer-assessment` - تقييم المراجع
- `PUT /api/reports/:id/admin-assessment` - تقييم المدير
- `PUT /api/reports/:id/spokesperson-assessment` - تقييم المتحدث
- `GET /api/reports/:id/full-review` - البيانات الكاملة
- `GET /api/external-links` - الروابط الخارجية
- `PUT /api/articles/:id` - تحديث مقالة
