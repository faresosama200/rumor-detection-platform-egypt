# 🚀 البدء السريع - نظام مواجهة الشائعات

## خطوات البدء في 5 دقائق

### 1️⃣ إعداد قاعدة البيانات
```bash
cd fake-news-backend
npm install
node database.js
```

### 2️⃣ تشغيل السيرفر الخلفي
```bash
npm start
# أو للتطوير مع إعادة تحميل تلقائية:
npm run dev
```
✅ السيرفر يعمل على: `http://localhost:5000`

### 3️⃣ تشغيل الواجهة الأمامية
```bash
cd frontend
npm install
npm run dev
```
✅ الموقع يعمل على: `http://localhost:5173`

## 📝 أول خطواتك

### للمستخدم العادي:
1. **انقر "رفع بلاغ"** من الرئيسية
2. **أكتب عنوان وتفاصيل الشائعة**
3. **أضف ملفات (صور، فيديو)**
4. **انقر "إرسال"** ✅

### للمراجع:
1. **اختر "فريق التحقق"**
2. **اختر بلاغ من القائمة**
3. **اضغط "عرض التفاصيل"**
4. **أضف تعليقك واختر الحالة**
5. **انقر "إرسال تقييم المراجع"** ✅

### للمدير:
1. **اختر "لوحة المدير"**
2. **مرر لأسفل للروابط الخارجية**
3. **أضف/عدل الروابط الموثوقة**
4. **انقر "حفظ الإعدادات"** ✅

### للمتحدث الإعلامي:
1. **اختر "فريق التحقق"**
2. **اختر بلاغ تم تحقيقه**
3. **أضف البيان الإعلامي الرسمي**
4. **اختر الحكم النهائي (صحيح/خاطئ)**
5. **انقر "نشر الحكم النهائي"** ✅
